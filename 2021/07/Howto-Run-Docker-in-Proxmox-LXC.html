一 26 七月 2021

如何在Proxmox的LXC容器中运行Docker环境

Step1

需要在PVE宿住机中加载以下两个模块 modprobe aufs modprobe overlay

同时修改/etc/modules文件，加入aufs,overlay两模块，实现每次开机自动加载模块

Step2

通过PVE的操作界面创建新容器 注意勾选“无特权的容器”

Step3

选项中的“签名”项，选择第一项“按键”与第二项“嵌套”，确定保存；

Step4

启动容器，登录后按常用方式安装Docker

Step5

运行Docker安装与运行是否正常

docker run hello-world成功，按以上方式可在PVE的LXC虚拟机中运行Docker服务。
